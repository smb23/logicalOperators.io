Evaluate whether each expression below will return TRUE or FALSE.

Assume that the value of x is 5.

Enter a value for x:
Expression Result Explanation
x == 6 Equality operator ( == ) checks if x is equal to 6.
x === "5" Strict equality operator ( === ) checks if x is equal to the string "5", which is not the same as the number 5.
x === 5 Strict equality operator ( === ) checks if x is equal to the number 5.
x != 5 Inequality operator ( != ) checks if x is not equal to 5.
x < 10 Less than operator ( < ) checks if x is less than 10.
x > 5 Greater than operator ( > ) checks if x is greater than 5.
x ≤ 5 Less than or equal to operator ( ≤ ) checks if x is less than or equal to 5.
x ≥ 5 && x != 10 Logical AND operator ( && ) checks if both expressions are true: x is greater than or equal to 5 AND x is not equal to 10.
x != 5 || x < 10 Logical OR operator ( || ) checks if at least one expression is true: x is not equal to 5 OR x is less than 10.
